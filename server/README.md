# A very Simple Shopping List e-commerce like REST API

## Functional requirement

- User (SignUp, Login,Delete)
- Product (Read All, Read by ID, Create, Update, Delete)
- Order (Read all, Read by ID, Create, Delete)

## Endpoints

### products

- /products
- /products
- /products/:productId
- /products/:productId
- /products/:productId

### orders

- /orders
- /orders
- /orders/:orderId
- /orders/:orderId

### user

- /user/signup
- /user/login
- /user/:userId
